Piraten-Tools
=================================================
Sammlung von Wordpress-Plugins für die Homepage der Göppinger Piraten

Die Module wurden aufgeteilt und finden sich jetzt getrennt unter:
https://github.com/PiratenGP/PT-nextpiratentreff
https://github.com/PiratenGP/PT-stimm
https://github.com/PiratenGP/PT-wikiimport

Ein Upgrade dürfte ohne Datenverlust möglich sein, einfach das alte Plugin deaktivieren und die neuen Plugins installieren und aktivieren.
